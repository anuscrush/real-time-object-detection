# Real-Time Object Detection System

## Overview
This project implements a real-time object detection system using YOLOv8 and OpenCV.

## Features
- Detects multiple objects in real-time
- Uses webcam/video input
- Displays bounding boxes with labels
- Saves output video

## Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- PyTorch

## Use Case
